# ZRO2-PropertyDoc v2.0.8

这是一个直接嵌在 ChatGPT 页面内的提示词工作台扩展，不需要依赖浏览器右上角弹窗来完成主操作。

## 核心能力

- 页面右下角悬浮入口，支持拖动、收起、展开
- 面板支持自由缩放，并记住位置与大小
- 提示词的新建、编辑、删除、保存
- 搜索标题、内容、标签
- 标签筛选与收藏筛选
- 收藏提示词并优先显示
- 点击整张提示卡直接复制
- 一键填入 ChatGPT 输入框
- 一键追加到当前输入框
- 空状态可直接新建第一条提示
- 标签建议补全
- JSON 导入
- 标准 JSON 导出
- 带元信息的本地备份
- 导入时支持合并去重或全部替换
- 点击整卡复制开关
- 重置窗口大小与位置

## 本次调整

- 收藏筛选移入搜索区，形成统一过滤器组
- 卡片动作重排，突出“填入 / 追加”，弱化编辑与删除

## 安装方式

1. 解压扩展 zip
2. 打开 Chrome 或 Edge 的扩展管理页
3. 开启“开发者模式”
4. 选择“加载已解压的扩展程序”
5. 选中本目录

## 使用提示

- 扩展在 `chatgpt.com` 和 `chat.openai.com` 页面加载
- 如果“填入 / 追加”没有生效，请先激活当前聊天输入框
- 建议定期使用“本地备份”保存完整库状态
