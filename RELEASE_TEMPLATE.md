# Release Template

Use the following template when creating a new GitHub Release.

---

## ZRO2 PromptDock vX.Y.Z

### 更新摘要

- 新增：
- 优化：
- 修复：

### 本版本包含

- 提示词管理能力更新
- 页面内交互体验调整
- 数据导入导出或兼容性修复

### 安装方式

1. 下载本次 Release 附件中的压缩包
2. 解压文件
3. 打开 Chrome 或 Edge 扩展管理页面
4. 开启“开发者模式”
5. 选择“加载已解压的扩展程序”
6. 选择解压后的目录

### 升级说明

- 建议升级前先导出当前提示词数据或执行本地备份
- 如果浏览器中已加载旧版本，建议先移除旧目录再重新加载新版本
- 若页面注入未立即生效，可刷新 ChatGPT 页面后重试

### 兼容说明

- 浏览器：Chrome / Edge
- 扩展清单：Manifest V3
- 目标页面：chatgpt.com / chat.openai.com

### 已知说明

- 当目标页面结构发生变化时，个别按钮定位或输入框注入行为可能需要后续版本适配

### 附件建议

- `ZRO2 PromptDock_vX_Y_Z.zip`
